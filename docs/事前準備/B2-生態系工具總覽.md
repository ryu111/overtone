## F. 趨勢觀察

### 六大趨勢

1. **SDD 成為共識** — GitHub/AWS/BMAD/Superpowers 都推「先規格後實作」
2. **Context Engineering 是核心技能** — CLAUDE.md/AGENTS.md/rules/ 都在解決 context 管理
3. **MCP 是主要擴展機制** — 比 hooks 或 commands 更活躍
4. **多 Agent 編排需求強** — Claude Squad/claude-code-mcp/VoltAgent 都指向協作
5. **格式標準收斂中** — AGENTS.md(60k+採用) + amplified.dev + CLAUDE.md(功能最強)
6. **企業快速跟進** — GitHub/MS/AWS/Google/Cloudflare 都有官方 MCP/Agent 投入

### 與 Overtone 定位對照

| 能力 | Overtone 已有 | 業界最佳 | 差距 |
|------|-------------|---------|------|
| 深度路由 D0-D4 | ✅ | BMAD 自適應 | 概念相當 |
| Skill 知識累積 | ✅ 28 個 | Superpowers ~20 | Overtone 更多 |
| instinct 進化 | ✅ 獨創 | 無對標 | 領先 |
| BDD 驅動 | ✅ | Kiro property-based | 概念相當 |
| rules/ 條件載入 | ❌ 未用 | Claude Code 原生 | 待採用 |
| MCP 整合 | 🟡 8 plugin | 生態豐富 | 可擴展 |
| 跨工具相容 | ❌ Claude only | AGENTS.md 60k+ | 不是目標 |
| 記憶管理 | ✅ state/timeline | Mem0/Letta | 概念不同 |
| Code review | 🟡 有 reviewer | CodeRabbit 2M repo | 可整合 |
