## 完整欄位分類

### 核心配置

| 欄位 | 說明 |
|------|------|
| `model` | 預設模型 |
| `availableModels` | 限制可選模型清單 |
| `modelOverrides` | 映射 model ID |
| `language` | 介面語言 |
| `outputStyle` | 輸出風格 |
| `alwaysThinkingEnabled` | extended thinking 開關 |

### 權限

| 欄位 | 說明 |
|------|------|
| `permissions.allow` | 自動允許的工具（glob 語法） |
| `permissions.deny` | 禁止的工具 |
| `permissions.ask` | 需要詢問的工具 |
| `permissions.defaultMode` | 預設權限模式（default/acceptEdits/plan/dontAsk/bypassPermissions） |
| `permissions.additionalDirectories` | 額外可讀目錄 |
| `permissions.disableBypassPermissionsMode` | 禁用 bypass 模式 |

**權限規則語法**：`Tool(specifier)` — 如 `Bash(npm run *)`, `Read(.env)`, `Edit(src/**)`, `Agent(name)`, `Skill(name *)`

### Sandbox

| 欄位 | 說明 |
|------|------|
| `sandbox.enabled` | 沙箱開關 |
| `sandbox.autoAllowBashIfSandboxed` | 沙箱內自動允許 Bash |
| `sandbox.filesystem.allowWrite/denyWrite/denyRead` | 檔案系統控制 |
| `sandbox.network.allowedDomains` | 網路域名白名單 |
| `sandbox.network.allowUnixSockets` | Unix socket 控制 |
| `sandbox.network.httpProxyPort/socksProxyPort` | Proxy 設定 |

**路徑前綴**：`//` = 絕對根、`~/` = home、`/` = settings 檔案位置、`./` = 相對

### Hooks

| 欄位 | 說明 |
|------|------|
| `hooks` | Hook 配置（見 [A6](./A6-Hook-系統.md)） |
| `disableAllHooks` | 關閉所有 hooks |
| `allowManagedHooksOnly` | 只允許 managed hooks |
| `allowedHttpHookUrls` | HTTP hook 白名單 |
| `httpHookAllowedEnvVars` | HTTP hook 可用環境變數 |

### MCP

| 欄位 | 說明 |
|------|------|
| `enableAllProjectMcpServers` | 自動啟用專案 MCP servers |
| `enabledMcpjsonServers/disabledMcpjsonServers` | MCP server 開關 |
| `allowedMcpServers/deniedMcpServers` | MCP server 白/黑名單 |

### Plugin

| 欄位 | 說明 |
|------|------|
| `enabledPlugins` | 啟用的 plugins |
| `extraKnownMarketplaces` | 額外 marketplace |
| `strictKnownMarketplaces` | 嚴格模式 |
| `blockedMarketplaces` | 封鎖的 marketplace |

### UI 與顯示

`showTurnDuration`, `statusLine`, `spinnerVerbs`, `spinnerTipsEnabled`, `spinnerTipsOverride`, `terminalProgressBarEnabled`, `prefersReducedMotion`

### Memory

`autoMemoryEnabled`, `autoMemoryDirectory`, `claudeMdExcludes`, `cleanupPeriodDays`

### Git/歸因

`attribution.commit`, `attribution.pr`, `includeGitInstructions`

### 環境

`env`（自訂環境變數）, `companyAnnouncements`

### 更新/實驗

`autoUpdatesChannel`, `fastModePerSessionOptIn`, `teammateMode`

### Managed-only 欄位

僅限 Managed Policy 層級設定：`allowManagedHooksOnly`, `allowManagedPermissionRulesOnly`, `allowManagedMcpServersOnly`, `network.allowManagedDomainsOnly`
